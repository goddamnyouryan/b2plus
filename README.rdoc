== B2Plus

= Getting started

1. Install Rails 3.2.13
2. Install Postgres
3. Add User 'sharpei' for postgres and grant all permissions
4. Run `bundle install`
5. Run `bundle exec rake db:create`
6. Run `bundle exec rake db:migrate`
7. Run `rails server`
8. Vist http://localhost:3000
9. Site should work!
